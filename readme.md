## 🎋 NFT 스토리

<br/>

### 1. 나무들의 산수화

현재 산수화의 의미는 틀렸다. 본래 산수화라는 것은 자연을 대상으로 그린 그림이 아니라, 나무들이 들어갈 수 있는 세상과 인간세상을 이어주는 문의 역할을 해주는 종이이다. 인간들을 싫어하는 몇몇의 나무들은 이 세상 속으로 들어갔고, 그렇지 않은 대다수의 나무들은 인간세상에 남았다.

<br/>

### 2. 인간들의 산수화

옛사람들은 그 종이를 발견하고, 예술작품으로 착각을 했다. 또한 자신이 그린 것처럼 종이의 이름을 붙이고 자신만의 예술 작품으로 만들었다. 이를 '산수화'라고 부르기 시작했고 지금의 의미가 되었다.

<br/>

### 3. 나무와 인간의 공존

인간들은 나무들이 감정을 가지고 있고 움직일 수 있다는 사실을 알지 못했다. 하지만 나무들은 이를 드러낼 필요가 없었다. 이전에 인간들은 나무들에게 잘 대해주었다. 나무들은 인간들과 함께 사는 것이 편했다. 나무들은 그런 인간을 위해 자신의 일부를 내어주며, 도움을 주었다.

<br/>

### 4. 인간들의 배신

어느순간 나무들은 살 곳이 없어졌다. 인간들은 나무들을 죽이고, 죽인 잔해로 인간들의 사리 사욕을 채우기 시작했다. 인간들은 나무들이 도망칠 수 없고 자신들의 재산이라고 생각했다. 하지만 나무들은 알고 있다. 산수화라는 자신들의 세상이 있다는 것을.....

<br/>

### 5. 나무들의 반란

나무들은 인간들을 피해 하나, 둘 점점 산수화 세상 속으로 뛰어들어가기 시작한다..... 인간들은 뒤늦게 이를 알아차렸다. 인간들은 나무들이 산수화 속으로 뛰어드는 행위를 '산수'라고 정의했다. 그리고 처음에 인간은 이를 막으려 노력했지만, 인간은 쉽게 변하지 않았다. 산수화로 먼저 들어간 나무들은 그곳에 정착하며, 인간들이 없는 자신들만의 세상을 만들어냈다. 그 후에 지구에 있는 모든 나무들에게 소리쳤다. '너희들의 세상이 준비됐다. 모두 산수하라!'

## 📆 NFT RoadMap

### 1ST : 민팅 및 홍보

- 민팅 후 10개 이상 구매한 홀더들에게 스페셜 산수화 NFT 에어드롭

<br/>

### 2ND 협업제품 제작

- 여러 기업들과 콜라보레이션 제품 제작 홀더 전용 오픈마켓 운영

<br/>

### 3RD 다양한 기부행사 개최

- NFT 10개 이상 보유한 홀더 이름으로 나무 심기 기부를 한다.

<br/>

### 4TH 한정판 NFT 발행

- 식목일, 크리스마스, 할로윈 등 기념일에 맞추어 한정판 NFT를 만들어 주기적으로 업데이트

<br/>

### 5TH 사업확장

- 산수화 컨셉 메타버스환경 구축 (미니게임 및 생활 시뮬레이션 가능)

<br/>
